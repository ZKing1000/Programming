DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/         2.5 or above
  PyGame:     http://www.pygame.org/         1.8.1 or above



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Press left mouse button and draw stroke (and release mouse button) this will propell the feather
in the corresponding direction of the stroke. Larger the stroke faster the feather moves. 

Crush the enemies using down-ward stroke and pick their color

Create new colors by combining primary colors

Left Click on the Feather when it is over a building you would like to paint.

Paint all the buildings and kill all the walking enemies (you do not need to kill the flying enemies) to complete the level.

In levels where you see flying enemies.. stay away from the scissors.
